1. 満局・待った・準備・席変更を event 化
今は本譜 move だけ event 化済みですが、それ以外の操作はまだ全 state が返ります。
特に準備OK、席変更、満局提案、待った申請/応答は小さい event で済みます。

- PC画面幅でのレイアウトを調整したい。
    - 画面縦幅に合わせる一方で、各ブロックに最大高さを設ける。盤面ブロックやチャットブロックが縦長すぎる。
    - 盤面と対局者・操作ボタンが離れてしまうのがよくない。盤面ブロック内は、各コンポーネントをくっつけて、盤とコンポーネントの間でなく、上下に余白をもたせるレイアウトにする。
- 投稿・入退室のタイムスタンプ表示は廃止する。
