playren

playren is a browser-based Renju game room. Players who enter the same room code join the same room; two players can play and other participants can watch.

Game Flow

  1. Enter a room
    Enter your display name and room code. The same room code connects users to the same room.
  2. Take a seat (Black / White)
    Use the + button on the Black or White seat. Participants can take an empty seat themselves.
  3. Mark ready
    When both seated players mark themselves ready, the game starts automatically.
  4. Opening (Taraguchi-10)
    The opening follows Taraguchi-10. During the opening, Swap choices and Black's Choice branch with 10 candidate moves may appear.
  5. Play the game
    After each move, time is added to the player's clock using Fischer timing.
  6. End and review
    The game ends by timeout, failure to reconnect in time, resignation, or agreed draw. After the game, you can review the move sequence and copy a RenjuPortal final-position kifu link from the game menu.

Main Features

  • Two-player games plus spectator seats
  • A player name ending with * indicates the host
  • The host can:
    • Set the clock (base minutes + increment seconds)
    • Manage player seats
  • Chat, including join/leave logs and comments
  • Draw offers, takebacks, and resignation
  • Position navigation (<<, <, Latest, >, >>, with arrow-key support)
  • Post-game review after the game ends
  • Kifu copy after the game ends, using a RenjuPortal final-position link
  • Display settings: language, sound, coordinates, board color, and move numbers
  • Reconnect support: the game pauses when a player disconnects. The player can return with the same display name and room code; if they do not return in time, the game ends.
