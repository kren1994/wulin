playren

playren 是可在浏览器中使用的 Renju（连珠）对局室。输入相同口令即可进入同一房间，2 人对局，其他参与者可以观战。

对局流程

  1. 进入房间
    输入昵称和口令后进入房间。口令相同的人会连接到同一个房间。
  2. 入座（黑 / 白）
    点击黑席或白席的 + 入座。座位空着时，参与者可以自行入座。
  3. 点击准备完毕
    两名入座者都点击准备完毕后，对局会自动开始。
  4. 开局（Taraguchi-10）
    开局按 Taraguchi-10 进行。过程中会出现 Swap 选择，以及黑方 Choice（10 题提示）分支。
  5. 进行对局
    每次落子后会增加持有时间（Fischer 规则）。
  6. 终局与复盘
    时间耗尽、未能在规定时间内重连、终局、同意和棋时，对局结束。结束后可以进入复盘模式前后查看手顺，也可以从聊天中的终局消息复制 RenjuPortal 的终局图棋谱链接。

主要功能

  • 2 人对局 + 观战席
  • 玩家名末尾显示 * 的人是主机
  • 主机拥有以下权限
    • 设置持有时间（基本分钟 + 每手加秒）
    • 管理对局席退席
  • 聊天（入退室记录 / 评论投稿）
  • 提和、悔棋、终局
  • 局面导航（<<, <, 最新, >, >>，支持方向键）
  • 复盘（仅终局后）
  • 复制棋谱（仅终局后，从聊天中的终局消息复制 RenjuPortal 的终局图链接）
  • 显示设置（语言切换、音效 ON/OFF、坐标显示 ON/OFF、棋盘色切换、手数显示 ON/OFF）
  • 重连支持（断线时会暂停并等待重连。使用相同昵称和相同口令可以返回房间；若在一定时间内未能返回，则终局）
